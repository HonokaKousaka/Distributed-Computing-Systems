## 题目描述:

* 现存在一份英文文本文件，要求统计文本中每个单次出现的次数，并将计数结果输出到文本中。

* 输入格式: 

  ```
  apple pen applepen pineapple pineapplepen boy cat dog boy cat
  dog cat dog dog
  ```

* 输出格式: 

  ```
  (pen,1)
  (pineapple,1)
  (dog,4)
  (pineapplepen,1)
  (apple,1)
  (applepen,1)
  (boy,2)
  (cat,3)
  ```
