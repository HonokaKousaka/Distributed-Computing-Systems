## 题目描述

* 要求：给定训练数据集和查询数据集，使用 KNN 算法完成对查询数据集的分类。（注意：当一条数据对应多个类别的权重相等时，请选择序号较小的类别。）

* 输入格式

  训练数据集：每行 n+2 个数字，分别是数据 id、类别 id、n 维特征，中间使用空格分割。

  ```
  1 0 1.0 0.9
  2 1 0.2 0.3
  3 0 0.8 0.7
  4 1 0.1 0.2
  5 0 0.8 0.8
  6 1 0.2 0.2
  ```

  查询数据集：每行 n+1 个数字，分别是数据 id、n 维特征，中间使用空格分割。

  ```
  7 0.8 0.9
  8 0.2 0.1
  ```

* 输出格式

  查询数据集类别：每行 2 个数字，分别是数据 id、类别 id，中间使用逗号分割。

  ```
  7,0
  8,1
  ```
