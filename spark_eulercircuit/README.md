### 题目描述：

- 辅导员让小明去各个学院搬书，问小明能否从本学院出发经过连接各个学院的路仅一次又再次回到本学院。

- 输入格式：

  输入包含若干行，每行两个数字a b,代表一条学院a到学院b的路,以空格分隔。

  ```
  1 2
  2 3
  3 4
  4 1
  ```

  

- 输出格式：

  问小明能否按照要求从本学院出发经过连接各个学院的路仅一次又再次回到本学院。可以则输出Yes,否则输出No。
  
  ```
  Yes
  ```
